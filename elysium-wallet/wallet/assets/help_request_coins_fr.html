En utilisant cette boîte de dialogue, vous pouvez recevoir des elysiums (vous devez avoir installé Elysium Wallet).

D'abord, saisissez le montant de Elysium à recevoir.

Ensuite, l'autre personne doit scanner le QR code généré. Ou bien, vous pouvez envoyer une URL Elysium à une personne à distance avec le bouton "publier" dans la barre d'action. L'autre personne devra cliquer l'URL Elysium.

Dasn les deux cas, l'autre personne se verra présenter une boîte de dialogue pour envoyer des elysiums avec tous les champs pré-remplis. Si elle accepte, elle n'a plus qu'à envoyer la transaction.
